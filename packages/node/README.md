## NPM Libaries

There is a npm library that can be used to interact with Deorg voting program.

## Installing

### Package manager

Using npm:

```bash
$ npm install @deorg/node
```

Using bower:

```bash
$ bower install @deorg/node
```

Using yarn:

```bash
$ yarn add @deorg/node
```

Using pnpm:

```bash
$ pnpm add @deorg/node
```
